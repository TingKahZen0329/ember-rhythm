# EmberRhythm 記憶節奏

這是一個使用 Python Tkinter 打造的間隔重複閃卡 (Spaced Repetition Flashcard) 應用程式，幫助使用者更有效率地學習與記憶單字。

## ✨ 主要功能

- [ ] 新增、修改、刪除單字卡片
- [ ] 根據 CEFR 等級 (A1-C2) 將單字分類
- [ ] 智能的間隔重複系統 (SRS)
- [ ] ...

## 🛠️ 技術棧

- Python 3
- Tkinter
- SQLite3

## 🚀 如何運行

1.  確保已安裝 Python 3。
2.  `git clone https://github.com/TingKahZen0329/ember-rhythm.git`
3.  `cd ember-rhythm`
4.  `python main.py`
